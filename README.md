# UI Storybook

Reusable react-typescript or anyweare type components

## Installation

Use the package manager npm or yarn to install foobar.

```bash
npm install
```

## Usage

```python
npm storybook
```

## Usage packages

in .npmrc add

- registry=https://npm.pkg.github.com/barbiture

```python
npm install @barbiture/skill-store-ui
```
